# PHP-Exercise-Basic
A collection of basic PHP exercises: variables, functions, arrays, looping, and conditionals.

## Questions
- Question 1: Simple User Profile
- Question 2: Calculating Grades
- Question 3: Mini Cashier System
- Question 4: Student Data & Grade Summary

## How to Run
1. Make sure PHP is installed
2. Run the file with the command:
php file_name.php
